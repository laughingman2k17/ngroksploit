# ngroksploit 

# Requirements
- Metasploit Framework
- Wine
- Mingw-w64 Compiler
- ngrok 
## ⭕️ Getting Started
. ``` apt-get install mingw-w64 ``
. ``` cd ngroksplit ``` 
. ``` ngrok tcp 22 or http 80  ```
. ``` python ngroksploit.py ```
. ``` dpkg --add-architecture i386 && apt-get update &&
apt-get install wine32 ```
